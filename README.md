# GOV.UK cookie example
Example of GOV.UK [cookie pattern](https://design-system.service.gov.uk/components/cookie-banner/) using Node.js and Hapi.js

## Prerequisites

Either:
- Docker
- Docker Compose

Or:
- Node 14+

## Running application
### Docker
```
docker-compose build
docker-compose up
```

### Node
```
node app
```
Or:
```
nodemon app
```
